# Meu Projeto Docker CRUD

Projeto CRUD usando Next.js (frontend), NestJS (backend) e MySQL, todo containerizado com Docker.

## Tecnologias

- Frontend: Next.js
- Backend: NestJS
- Banco de dados: MySQL
- Docker e Docker Compose para orquestração
